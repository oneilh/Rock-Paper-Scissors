# Rock Paper Scissors

![React](https://img.shields.io/badge/React-18.3-blue) ![Vite](https://img.shields.io/badge/Vite-5.4-purple) ![Status](https://img.shields.io/badge/Status-Active-success)

A modern, interactive frontend implementation of the classic Rock-Paper-Scissors game.

## 📖 About The Project

This project is an incourse frontend application demonstrating core React concepts, responsive UI design, and client-side state management. Adapted from a beautiful [Dribbble design](https://dribbble.com/shots/13946587-Rock-Paper-Scissors-game), it serves as a robust, production-ready example of handling game logic and component architecture in React.

## ✨ Key Features

- **Interactive Gameplay**: Play the classic game against a randomized computer opponent.
- **Score Management**: Persists and tracks your score across multiple rounds.
- **Responsive UI**: A polished, engaging interface optimized for both desktop and mobile viewing.

## 🛠 Tech Stack

- **React 18** - UI Library
- **Vite** - Build Tool & Development Server
- **React Router** - Client-side Routing
- **Vanilla CSS** - Styling
- **React Icons** - SVG icon assets

## 🏗 Architecture & Implementation Details

- **State Management**: React Hooks (`useState`, `useEffect`) manage game flow, score persistence, and player selections.
- **Component Design**: Modular, reusable UI components for game elements (buttons, scoreboards) ensuring clean and maintainable code.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
- Node.js installed on your machine

### Installation

1. **Clone the repo**
   ```sh
   git clone https://github.com/your-username/rock-paper-scissors.git
   ```
2. **Navigate into the directory**
   ```sh
   cd Rock-Paper-Scissors
   ```
3. **Install NPM packages**
   ```sh
   npm install
   ```
4. **Start the development server**
   ```sh
   npm run dev
   ```
   *The app will be running at `http://localhost:5173`.*
